# ♟️ Chess App (C++ / Raylib)

A lightweight **C++ chess application** that implements the **full rules and logic of chess** with a graphical interface built using **Raylib**.

The project focuses on implementing chess mechanics from scratch while keeping the code relatively small and understandable.

## Features

* ♟️ Full chess rules implementation
* ✅ Legal move validation
* 🔁 Turn-based gameplay
* ♕ Piece movement logic for all pieces
* 🖥️ Graphical window using Raylib
* 💻 Written entirely in C++

## Technologies

* **C++**
* **Raylib** – window creation and rendering

## How It Works

The application contains a full internal representation of a chess board and implements the rules of chess manually, including:

* Piece-specific movement
* Move legality checks
* Detection of check situations
* Enforcement of valid gameplay

Rendering and input handling are done through **Raylib**, which provides the game window and drawing utilities.

## Goals of the Project

* Practice **C++ architecture and logic design**
* Implement **chess rules from scratch**
* Build a simple **graphical chess interface**
